# **Самокат**
Проект включает анализ продаж по товарным категориям и подкатегориям.
## Основные задачи:
1. Определение самых ходовых товарных групп
2. Расчет среднего чека
3. Оценка доли промо-акций
4. Расчет маржи по категориям
5. ABC-анализ 
## Инструменты
1. Python (pandas, numpy)
2. Matplotlib, seaborn для визуализации
## Результат
В результате анализа выявлены ключевые товарные категории, рассчитана маржинальность и проведена ABC-классификация.
